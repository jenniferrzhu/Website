Birthday Website Template

Celebrate any friend's birthday by building a website filled with memories to celebrate. Template features include a lock screen, a timeline show through pictures, blog pages per picture, a playful question page, and a letter.

Built with React.ts.

Geospatial Habitat Analysis

Identifying areas at risk of habitat loss in Lake Tahoe by comparing wildlife and urbanization data. Analyzed density-based clusters via the elbow method and K-means clustering. Concluded areas with low intensity urbanization (i.e. housing, commerce, etc.) would interfere with wildlife the most.

Build with Python and the following libraries: Scikit-Learn, Rasterio, Matplotlib.

OpenUp (HackMIT)

Social media application with daily thought prompts to connect friends on a deeper level. After answering the prompt page, the rest of the features unlock, including a page to view responses from friends with a heart per day maximum to add value to the like for their favorite response.

Backend: Express, MongoDB, Joi, JavaScript Frontend: React Native, JavaScript, React Navigation, Expo Go
